This mod/datapack uses the mannequin entity as a corpse for when the player dies! ✨

Corpses 💀
It makes it so that instead of players dropping their items around randomly, they will now leave behind a corpse with their skin and name from which they can later retrieve their stuff.

To retreieve the items from the corpse they must first locate it at the place where they last died and sneak while being near it, the items should pop back in their inventory and the corpse should disappear.

There is no despawn timer unless added via the config screen

Coprse Finder ⚙️
The Corpse Finder Screen allows the player to see all the currently loaded corpses and their coordinates

Can be accessed by typing: /trigger corpse_finder

Configeration Screen ⚙️
The Config Screen offers many options for players to choose how they want the pack to work to match their playstyle.

Can be accessed by typing: /trigger config
